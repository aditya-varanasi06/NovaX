# NovaX 🚀

A sleek, modern, and fully responsive landing page built with pure HTML and CSS — no frameworks, no dependencies.

---

##  Features

- **Animated Gradient Hero** — Smooth infinite background animation using CSS `@keyframes`
- **Custom Cursor** — A glowing cyan ring replaces the default cursor for a futuristic feel
- **Sticky Glassmorphism Navbar** — Frosted-glass header that stays fixed on scroll
- **Hover Underline Links** — Animated underline effect on navigation links
- **Scroll Reveal Cards** — Feature cards animate in with a fade-and-rise effect
- **Fade-in Hero Text** — Staggered entrance animations for headline, subtext, and CTA button
- **Fully Responsive** — Adapts seamlessly to mobile, tablet, and desktop screens

No build tools or installations required.

1. **Clone or download** the repository
2. Open `index.html` in any modern browser

```bash
git clone https://github.com/your-username/novax.git
cd novax
open index.html
```

##  Design Highlights

| Element | Detail |
|---|---|
| Color Palette | `#00d4ff` cyan, `#0f2027` dark navy, `#111` charcoal |
| Font | Segoe UI (system font) |
| Hero Background | 4-stop gradient animated at 400% size |
| Card Style | Glassmorphism — `backdrop-filter: blur` + semi-transparent background |
| Border Radius | Rounded (`20px` cards, `25px` buttons) for a modern look |

---

##  Responsive Behavior

| Breakpoint | Behavior |
|---|---|
| `> 768px` | 3-column feature cards, full navbar |
| `≤ 768px` | Stacked single-column cards, navbar links hidden |
